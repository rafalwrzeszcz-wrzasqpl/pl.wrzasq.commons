About ChillDev Commons

ChillDev-Commons is general-purpose library for Java.

Project Modules

This project has declared the following modules:

Name Description
ChillDev Commons AWS Helper components for working with Amazon Web Services.
ChillDev Commons Client Web services client building utilities.
ChillDev Commons Daemon Helper components for handling daemon services with Apache Commons Daemon.
ChillDev Commons Data Helper routines for working with Spring Data structures.
ChillDev Commons DB Base database and model-related structures and routines.
ChillDev Commons JWT JSON Web Tokens utilities.
ChillDev Commons Text Text processing utilities.
ChillDev Commons Web Web-app related utilities.
